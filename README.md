# regressao-bnds-atividade
Análise de regressão sobre operações de financiamento não automáticas do BNDES. Projeto prático para a disciplina de Ciência de Dados do IFMA, contemplando limpeza de dados, análise exploratória, regressão simples e múltipla com métricas de performance (R², MSE). Aluno: Jose Fanini.
